The content of this website is written in Markdown and the site's HTML pages are
generated using the [Zola](https://www.getzola.org/) static site generator. The
website's layout and style are controlled fundamentally using the
[PicoCSS](https://picocss.com/) CSS framework with custom CSS code to overrride
the defauls as needed. There is no particular style or colour theme, but the
colours are based on the
[Gruvbox](https://www.color-hex.com/color-palette/1026676) colour palette.
